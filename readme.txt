== Description ==
**IMPORTANT: *WooCommerce SMS Notifications* requires WooCommerce 2.1.0 or higher.**

**SMS API: Buy SMS credits on [Jakewer](http://www.jakewer.com)**


**WooCommerce SMS Notifications** Sends SMS notifications to your clients for order status changes. You can also receive an SMS message when a new order is received.
 
= Features =  
* Supported SMS gateway: Jakewer(http://www.jakewer.com)
* Notify store owner about new orders
* Send SMS notification to client for each order status change
* Customize SMS template for each order status
* Custom signature can be appended to all outgoing SMS
* Support for a large number of standard variables in SMS templates
* Option to add custom variables in SMS templates
* Send international SMS
* Send SMS to billing or shipping address phone
* Automatically insert the international country code prefix to the customerâ€™s phone number
* Can log all outgoing SMS for audit trail

== Installation ==
1. You can:
 * Upload the `su-wc-sms-notifications` folder to `/wp-content/plugins/` directory via FTP. 
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **WooCommerce SMS Notifications** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Set up plugin on *WooCommerce -> WooCommerce SMS Notificationss* or through *Settings* link.
4. For obtaining credentials, please register at (http://www.jakewer.com/my-account/)
